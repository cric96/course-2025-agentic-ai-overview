# Agentic AI 
## Overview e focus su MCP

Questo blocco di slide offre una panoramica introduttiva sull’**agentic AI** e sul suo impatto nello sviluppo del software. In particolare, vengono affrontati:

- i concetti chiave di agenti, uso di strumenti (*tool use*) e orchestrazione
- le differenze tra semplici LLM e sistemi agentici più complessi
- esempi di workflow agentici applicati alla programmazione
- limiti attuali, rischi e buone pratiche di utilizzo in contesti reali
